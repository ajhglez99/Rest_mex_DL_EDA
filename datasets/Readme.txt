## English
The data set is completely in the rest-mex_2021_Sentiment_data_train.csv file
The data set consists of 5197 lines (70% of the original data set. The other 30% will be used as a test set), one for each opinion. Each row contains nine columns:
1. Index: It is the index of each opinion. Data type: Integer
2. Title: The title that the tourist himself gave to his opinion. Data type: Text
3. Opinion: The opinion expressed by the tourist. Data type: Text
4. Place: The tourist place that the tourist visited and to which the opinion is directed. Data type: Text
5. Gender: The gender of the tourist. Data type: [Male, Female, N / I] (N / I = Not Information)
6. Age: The age of the tourist at the time of issuing the opinion. Datatype; Integer (-1 = Not Information)
7. Country: Country of origin of the tourist: Type of data: Text (In Spanish) (N / I = Not Information)
8. Date: Date the opinion was issued. Data type: Date
9. Label: The label that represents the polarity of the opinion. Data type: [1, 2, 3, 4, 5]

The polarity ranges from 1, which means the highest degree of dissatisfaction, to 5, which is the highest degree of satisfaction. It can be interpreted as follows:
1. Very bad
2. Bad
3. Neutral
4. Good
5. Very good

## Spanish (Español)
El conjunto de datos se encuentra completamente en el archivo rest-mex_2021_Sentiment_data_train.csv
El conjunto de datos consta de 5197 renglones (el 70 % del conjunto de datos original. El otro 30 % será utilizado como conjunto de prueba), uno por cada opinión. Cada renglón contiene 9 columnas:
1. Index: Es el índice de cada opinión. Tipo de dato: Entero
2. Title: El título que el propio turista le otorgó a su opinión. Tipo de dato: Texto 
3. Opinion: La opinión emitida por el turista. Tipo de dato: Texto
4. Lugar: El lugar turístico que visitó el turista y al cual va dirigida la opinión. Tipo de dato: Texto
5. Género:  El género del turista. Tipo de dato: [Male, Female, N/I] (N/I = Not Information)
6. Edad: La edad del turista al momento de emitir la opinión. Tipo de dato; Entero ( -1 = Not Information)
7. País: País de origen del turista: Tipo de dato: Texto (En español) (N/I = Not Information)
8. Fecha: Fecha en la que se emitió la opinión. Tipo de dato: Fecha
9. Etiqueta: La etiqueta que representa la polaridad de la opinión. Tipo de dato: [1, 2 ,3 ,4 ,5]

La polaridad va de 1 que significa el mayor grado de insatisfacción, hasta 5 que es el mayor grado de satisfacción. Se puede interpetar de la siguiente manera:
1. Muy malo
2. Malo
3. Neutral
4. Bueno
5. Muy bueno
